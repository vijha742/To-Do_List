# To-Do List
This to do list project written in java is aimed to learn the ins and outs of java through this project.

## Technologies used:
- Java 11
- GSON
- Tmux
- Bash script

## Feature available
- Add a task
- Delete a task
- Mark task as done
- Edit task
- Display task
More features updating soon...

### Structure of task
- Title
- Due date
- Task id (Not yet available)
- Status
- Description
